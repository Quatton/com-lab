理学部情報科学科　J4220661
タエチャスクナッチャノン

まずは、
maximum recursion depth exceeded
が出てくる。

これは、再帰の深さが深すぎるということである。
ただし、10が大きすぎるためではなく、(fact3 0)の場合も同様に出てくる。

(fact3 0)の場合、
(if-fun (< 0 1)
        1
        (* 0 (fact3 (- 0 1))))

if-fun は if と違って、
末尾が完全に終わって再帰しなければ評価されないため、
(< 0 1)が真であること、(* 0 (fact3 (- 1 1)))が 0であることにもかかわらず、
(fact3 -1), (fact3 -2), ... が終わらなければ、再帰しない。
